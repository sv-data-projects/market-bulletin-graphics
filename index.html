SV Recovered Resources Market Bulletin graphics index

This page contains links to HTML/SVG/JS based data visualisations for embedding into various sections of online version of the SV Recovered Resources Market Bulletin. Each of these visualisations will automatically update when data is updated in the supporting data tables (hosted in a Google Sheet) here. This Google Sheet also contains some user guidance documentation. However all technical documentation can be found in the readme.ds file of the GitHub repository
