SV Recovered Resources Market Bulletin graphics index

This page contains links to HTML/SVG/JS based data visualisations for embedding into various sections of online version of the SV Recovered Resources Market Bulletin. Each of these visualisations will automatically update when data is updated in the supporting data tables (hosted in a Google Sheet) here. This Google Sheet also contains some user guidance documentation, including a tool for generating URLs and iframe links (for CMS component creation), with date query strings. However all technical documentation can be found in the readme.md file of the GitHub repository

The following links are grouped by pages in the draft HTML version of the Market Bulletin, as built in Craft CMS (note: link available at time of writing in mid-September, 2021 and should be updated once a production link is established)

Notes:

  • Value chain price diagrams include all sub-material prices Instructions to limit/hide and reposition are in the documentation
  • Documentation for adapting the bin composition graphic (future need) is included in the github and GSheet documentation
